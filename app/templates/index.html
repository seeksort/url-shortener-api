URL Shortener API

This microservice will shorten URLs for you when passed into this site's URL. Then, it will redirect you to your original URL if you pass in the shortened URL code into this site's URL.

Creating a Short URL

In your search bar, add your URL to be shortened after /new/. Example:

https://REPLACE_URL.herokuapp.com/new/https://www.freecodecamp.org/

Output

{
"original_url": "https://www.freecodecamp.org",
"short_url": "https://REPLACE_URL.herokuapp.com/1001"
}

Using a Short URL

In your search bar, add your new shortened URL to be shortened after /. Example:

https://REPLACE_URL.herokuapp.com/1001

Redirects you to

https://www.freecodecamp.org

\o/
